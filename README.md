# AI Logic Audit Reports

AI生成コードにおける論理破綻を専門に調査・監査するプロジェクトです。
AIは「文法」は書けますが、「ビジネスの論理（ビジネスロジック）」まで完璧には理解していません。本リポジトリでは、AI生成コードに潜む論理的な欠陥（状態遷移の不整合、競合状態、脆弱性）を可視化・修正します。

## Audit Methodology
本リポジトリでは、AI開発におけるデバッグの階層構造（Context Ladder）の「Rung 4（期待値と論理的意図の明確化）」を監査基準として採用しています。

AIが生成したコードの断片的な不具合（文法エラー）だけでなく、ビジネスロジックの「意図」まで掘り下げ、状態遷移の不整合や論理的な脆弱性を網羅的に診断・修正します。
